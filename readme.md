Emotion understanding from short textual thoughts is an important task in Natural Language Processing (NLP) because human communication often contains implicit emotional signals. This project proposes a deep learning based emotion classification system that predicts the emotional state expressed in a short piece of text (thought). The system uses DistilBERT, a lightweight transformer architecture derived from BERT, to efficiently capture contextual meaning in sentences while maintaining high performance with reduced computational cost.

The model is fine-tuned on the HuggingFace Emotion dataset, which contains labeled text samples belonging to six emotional categories: sadness, joy, love, anger, fear, and surprise. During preprocessing, the input text is tokenized using the DistilBERT tokenizer, which converts raw text into token IDs and attention masks required by the transformer model. The tokenized input is then passed through the pretrained DistilBERT encoder, which generates contextual embeddings representing semantic relationships between words in the sentence.

The final hidden representation corresponding to the sentence is fed into a classification head consisting of a linear layer followed by a softmax activation function, which produces probability scores for each emotion class. The model is trained using cross-entropy loss, enabling it to learn the correct mapping between textual patterns and emotional labels.

During inference, the system receives a user thought as input, processes it through the tokenizer and the trained DistilBERT classifier, and outputs both the predicted emotion label and a confidence score representing the model's certainty. The performance of the model is evaluated using multiple metrics including accuracy, precision, recall, F1 score, and confusion matrix, providing a comprehensive understanding of classification performance.

This architecture enables accurate emotion detection while remaining computationally efficient due to the compact design of DistilBERT. The model can be integrated into real-time applications such as emotion-aware social platforms, mental health monitoring tools, conversational agents, and sentiment-aware recommendation systems.
